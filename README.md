### 進め方
問題は全てPythonで解いていきます。<br>
1周目は星4以下のA問題を解いていき、1章終わるごとに、その章で解けなかった問題をもう一度解きます。<br>
2周目で、1周目解けなかった問題+星5以上+B問題を解いていきます。<br>
また、ABCのために本書を進めているので、7章のヒューリスティックは飛ばして、必要なときに読みます。
